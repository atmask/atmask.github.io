Enabling experimental mode on Nix

Overview Here’s a short one! Ever run into the following with Nix? error: experimental Nix feature 'nix-command' is disabled; add '--extra-experimental-features nix-command' to enable it Let’s talk about it Flakes are experimental? So flakes are experimental? Yes, in Nix flakes are still gated as experimental. Want to get around it quickly? nix --extra-experimental-features 'nix-command flakes' flake check For a longer term ease of use you are better off just enabling this permanently: mkdir -p ~/.config/nix && \ echo 'experimental-features = nix-command flakes' >> ~/.config/nix/nix.conf
